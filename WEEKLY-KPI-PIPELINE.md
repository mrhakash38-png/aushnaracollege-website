## 4. Data Collection Process

### Google Search Console:
1. Navigate to: https://search.google.com/search-console
2. Select property: `aushnaracollege.edu.bd`
3. Go to **Performance** ‚Üí Set date range to "Last 7 days"
4. Export data: **Export ‚Üí Google Sheets** (or CSV)
5. Copy key metrics to weekly report

### Bing Webmaster Tools:
1. Navigate to: https://www.bing.com/webmasters
2. Select site: `aushnaracollege.edu.bd`
3. Review **Site Performance** dashboard
4. Note any significant trends or changes

### Google Analytics (GA4):
1. Navigate to: https://analytics.google.com
2. Select property: `Aushnara College`
3. Go to **Reports ‚Üí Engagement ‚Üí Overview**
4. Set date range to "Last 7 days" with "Compare to previous period"
5. Export key metrics
